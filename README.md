### 📍 Installing the dependencies
`npm i` | `npm install`

### 🔧 Setup

Make a **config.json** file in the root directory of the project and add:

```
{
  "prefix": "!",
  "ownerID": "OWNER-ID-HERE",
  "token": "TOKEN-HERE"
}
```

### 📰 Commands | Category: Template

You can add both as categories and as also commands:

| Command   | Description                         | Usage                   |
| --------- | ----------------------------------- | ----------------------- |
| !avatar   | Get a user's avatar                 | !avatar @x4slow         |
| !water    | Get an image with a custom text     | !water pepsi cola       |
| !test-message | Respond with a "test" message (really test command)   | !test-message                |
